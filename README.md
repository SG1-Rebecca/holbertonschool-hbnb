## 📖 Description


A simplified version of an AirBnB-like application, named **HBnB Evolution**.  
The application allows users to perform the following primary operations:

- 👤 **User Management** — Register, update profiles, and manage roles (user / admin).
- 🏠 **Place Management** — List properties with name, description, price, location, and amenities.
- ⭐ **Review Management** — Leave ratings and comments on visited places.
- 🛎️ **Amenity Management** — Manage amenities that can be associated with places.
